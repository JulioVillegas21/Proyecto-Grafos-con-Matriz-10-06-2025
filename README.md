# Proyecto Grafos con Matriz

Este proyecto fue desarrollado como **Trabajo Práctico** para la materia **Algoritmos y Estructuras de Datos II**

Fecha: 10/06/2025

## Descripción
Aplicación en Java que implementa un **grafo utilizando matriz de adyacencia**. Permite representar relaciones entre vértices y trabajar con estructuras de grafos.

## Funcionalidades principales
- Creación de vértices
- Representación de conexiones mediante matriz
- Visualización del grafo
- Gestión de relaciones entre nodos

## Características técnicas
- Uso de matriz de adyacencia
- Representación estructurada de grafos
- Manipulación básica de nodos y conexiones

## Estructura del proyecto
- `GrafoMatriz`: implementación del grafo  
- `Vertice`: representación de los nodos  
- `Main`: ejecución del programa  

## Tecnologías
- Java

## Ejecución
Compilar y ejecutar la clase `Main`.
